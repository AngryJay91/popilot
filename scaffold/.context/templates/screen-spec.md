## Level 1. 전체 IA & 진입점

> 이 기능이 전체 서비스 구조에서 어디에 위치하는지 정의

### 1.1 정보 구조 (IA)

```
[상위 메뉴] (GNB/LNB)
├── [기존 메뉴 1]
│   └── (기존 구조)
│
├── [기존 메뉴 2]
│   └── (기존 구조)
│
└── [신규/변경 기능] ← 이 화면정의서의 범위
    ├── [하위 요소 1]
    │   ├── [세부 요소 a]
    │   └── [세부 요소 b]
    └── [하위 요소 2]
```

|--------|------|------------|------------|------|
| **A** | [위치 상세] | [클릭/탭 등] | [화면 이동/모달 오픈 등] | [우선순위/빈도] |

| **C** | [위치 상세] | [클릭/탭 등] | [화면 이동/모달 오픈 등] | |

### 1.3 메뉴 위치 정책

| 항목 | 정책 | 근거 |
|------|------|------|
| **메뉴 위치** | [GNB/LNB/탭 등] | [사용자 접근성, 정보 구조 등] |

### 4.2 [모달명 1] 상세

#### 4.2.1 전체 레이아웃

```
┌─────────────────────────────────────────────────────────────────┐
│                                                           [✕]   │
│  [모달 타이틀]                                                  │
│                                                                 │
│  ─────────────────────────────────────────────────────────────  │
│                                                                 │
│  [섹션 1 라벨]                                                  │
│  ┌─────────────────────────────────────────────────────────────┐│
│  │ [입력 필드 / 선택 UI / 정보 표시]                           ││
│  └─────────────────────────────────────────────────────────────┘│
│                                                                 │
│  [섹션 2 라벨] (선택)                                           │
│  ┌─────────────────────────────────────────────────────────────┐│
│  │ [입력 필드]                                                 ││
│  └─────────────────────────────────────────────────────────────┘│
│  0/[최대글자수]                                                 │
│                                                                 │
│  ─────────────────────────────────────────────────────────────  │
│                                                                 │
│  ☐ [체크박스 옵션]                                              │
│                                                                 │
│  ─────────────────────────────────────────────────────────────  │
│                                                                 │
│                                     [취소]  [확인]              │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

#### 4.2.2 섹션별 상세

| 섹션 | 요소 | 타입 | 스펙 | 필수 |
|------|------|------|------|------|
| 헤더 | 타이틀 | 텍스트 | "[타이틀]" | - |
| 헤더 | 닫기 | 버튼 | [✕] | - |
| [섹션1] | [필드명] | [input/select/textarea 등] | [placeholder, 기본값] | ✅/❌ |
| [섹션2] | [필드명] | [타입] | [스펙] | ✅/❌ |
| 옵션 | [체크박스명] | checkbox | 기본값: [체크/미체크] | - |

#### 4.2.3 유효성 검증

| 필드 | 규칙 | 에러 메시지 |
|------|------|------------|
| [필드명] | 필수 | "[필드명]을(를) 입력해주세요" |
| [필드명] | 숫자만 | "숫자만 입력해주세요" |
| [필드명] | 범위 [N~M] | "[N]~[M] 사이 값을 입력해주세요" |
| [필드명] | 최대 [N]자 | (입력 자체 제한) |

#### 4.2.4 저장 후 동작

| 단계 | 동작 |
|------|------|
| 1 | API 호출 ([METHOD] /[endpoint]) |
| 2 | 로딩 스피너 표시 |
| 3 | 성공 시: 모달 닫기 |
| 4 | 성공 시: 토스트 "[메시지]" |
| 5 | 성공 시: [후속 동작: 목록 새로고침 등] |
| 6 | 실패 시: 에러 토스트 + 모달 유지 |

### 4.3 삭제 확인 다이얼로그

> 삭제/취소 등 위험 액션 확인용
