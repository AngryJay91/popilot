# S-01: 칸반 보드

## AC (Acceptance Criteria)

1. Given 태스크가 3개 이상 존재할 때
   When 페이지를 로드하면
   Then Todo / In Progress / Done 3개 컬럼에 태스크 카드가 상태별로 분류되어 표시된다

2. Given In Progress 컬럼에 카드가 있을 때
   When 카드를 Done 컬럼으로 드래그하면
   Then 카드가 Done 컬럼으로 이동하고 localStorage에 즉시 반영된다

3. Given 각 카드에 담당자와 우선순위가 지정되어 있을 때
   When 카드를 확인하면
   Then 담당자 아바타(이니셜)와 우선순위 뱃지(색상)가 카드에 표시된다

## Data Model

```typescript
type TaskStatus = 'todo' | 'in-progress' | 'done'
type Priority = 'low' | 'medium' | 'high' | 'critical'

interface Task {
  id: string
  title: string
  description: string
  assignee: string
  priority: Priority
  status: TaskStatus
  createdAt: string
}
```
