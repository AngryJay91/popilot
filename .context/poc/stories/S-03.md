# S-03: 필터 & 요약

## AC (Acceptance Criteria)

1. Given 보드 상단에 요약 바가 표시될 때
   When 태스크가 Todo 3개, In Progress 2개, Done 1개이면
   Then 각 컬럼 카운터에 3, 2, 1이 표시된다

2. Given 담당자 필터 드롭다운이 있을 때
   When "Bob"을 선택하면
   Then Bob에게 할당된 태스크만 보드에 표시되고, 카운터도 필터된 수로 갱신된다

3. Given 필터가 "Bob"으로 적용된 상태에서
   When "전체" 옵션을 선택하면
   Then 모든 태스크가 다시 표시된다
