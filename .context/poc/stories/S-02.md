# S-02: 태스크 생성/편집

## AC (Acceptance Criteria)

1. Given 칸반 보드가 표시된 상태에서
   When "새 태스크" 버튼을 클릭하면
   Then 모달이 열리고 제목, 설명, 담당자, 우선순위 입력 폼이 표시된다

2. Given 모달에서 필수값(제목)을 입력하고 저장 버튼을 누르면
   When 저장이 완료되면
   Then 모달이 닫히고 새 태스크가 Todo 컬럼에 즉시 나타난다

3. Given 칸반 보드에서 기존 카드를 클릭하면
   When 편집 모달이 열리면
   Then 기존 값이 폼에 채워져 있고, 수정 후 저장하면 카드에 반영된다
